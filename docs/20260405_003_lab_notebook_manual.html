LAB//NOTEBOOK

A field engineering notebook for electronics design, testing, and documentation. Cross-device. Mobile-first. No build tools required.

VERSION 1.0 — 2026-04-05 — AM_receiver_20260405

01 Overview

LAB//NOTEBOOK is a browser-based engineering workbench. It runs directly from disk or GitHub Pages — no server, no framework, no installation. Open the HTML file in any browser on any device and it works.

The application is organized around projects. Each project gets its own folder with its own schematic, BOM, and log files. Common assets — the component library, editor engine, shared styles — live closer to the root and are shared across all projects.

Design Principle This tool is built for the bench, not the boardroom. It prioritizes speed of use, offline-first operation, and data portability. Your data is always plain JSON — readable, scriptable, version-controllable.

The status bar at the bottom shows current tool, zoom level, cursor position, component count, and wire count.

Mobile On small screens the Properties panel is hidden to maximize canvas space. Access component properties via right-tap (long press) → Edit Value.

03 Your First Schematic

Open 20260405_001_schematic_editor.html in your browser.

In the Component Library (left panel), find the component you need. Use the search box to filter.

Drag the component onto the canvas. It snaps to the grid automatically. On mobile, tap the component to place it at canvas center, then drag it into position.

Click the component to select it. Set its value in the Properties panel (right) — e.g. 220µH for an inductor.

Press W or click WIRE in the toolbar. Click a pin (orange dot) to start a wire. Click another pin or grid point to complete it.

Press Escape to cancel an in-progress wire. Press V to return to Select mode.

Click EXPORT JSON to save your work. The file is named with today's date automatically.

04 Select Tool

Activated by V or clicking SELECT in the toolbar.

Click to select

Click any component to select it. The component highlights in cyan. Its properties appear in the right panel. Click empty canvas space to deselect.

Drag to move

Click and drag a selected component to reposition it. Movement snaps to the 20px grid. Connected wires do not move with the component — rewire after repositioning.

Pan canvas

Click and drag on empty canvas space to pan the view. Middle-mouse button also pans.

Rotate and flip

With a component selected: press R to rotate 90° clockwise. Press F to flip horizontally. Both are also available in the top bar and right-click context menu.

05 Wire Tool

Activated by W or clicking WIRE in the toolbar. The cursor changes to a crosshair.

Drawing a wire

Click to set the start point. Move the mouse — the wire routes orthogonally (horizontal then vertical). Click to place the end point. The wire is committed.

Pin snapping

When the cursor is within 15 pixels (screen space) of a component pin, it snaps to that pin automatically. The orange pin dot becomes the wire endpoint.

Canceling

Press Escape to cancel the current wire in progress without placing it.

Note Wires are independent objects. Moving a component after wiring it does not move its connected wires. Delete and redraw wires after repositioning components.

06 Delete Tool

Activated by D or clicking DELETE in the toolbar.

Click any component to delete it immediately. Click near a wire to delete that wire segment. There is no undo — export JSON before making large deletions.

In Select mode, pressing Delete or Backspace deletes the currently selected component.

08 Component Library

The library panel lists all available components organized by category. Type in the search box to filter by name or type string.

Categories: PASSIVES, SEMICONDUCTORS, ICs, POWER, RF.

09 Placing Components

Drag and drop (desktop)

Drag a component from the library panel onto the canvas. It places at the drop position, snapped to the nearest grid point.

Tap to place (mobile)

Tap any component in the library. It places at the center of the current canvas view. Drag it into position from there.

Duplicate

Right-click (or long-press) a placed component → Duplicate. The copy appears offset by two grid squares. Useful for placing multiple identical components.

Reference designators

References are assigned automatically in sequence: R1, R2, C1, C2, L1, U1, etc. You can rename any reference in the Properties panel or via right-click → Edit Label.

12 Drawing Wires

Wires route orthogonally — horizontal first, then vertical, based on the dominant axis of movement. This keeps schematics clean without manual routing decisions.

To force a vertical-first route, start moving vertically before moving horizontally.

13 Junctions

When three or more wires meet at the same point, a filled dot (junction marker) appears automatically. No manual placement needed. Two crossing wires with no junction are not electrically connected.

14 Pin Snapping

In Wire mode, orange dots appear on component pins. The cursor snaps to the nearest pin within 15 screen pixels. When snapped, the wire endpoint locks to the exact pin coordinate.

In Select mode, pins are shown on the selected component only, as a placement guide.

15 Zoom & Pan

Zoom

Scroll wheel zooms toward the cursor position. ZOOM+ and ZOOM− buttons in the toolbar step zoom by 20%. FIT fits all placed components into view.

Zoom range: 10% to 800%.

Pan

Click and drag on empty canvas space. Middle mouse button drag. Alt + left drag.

17 Export JSON

Clicking EXPORT JSON downloads a .json file containing the complete schematic state: all components (position, type, reference, value, rotation, flip, notes) and all wires.

The file is named automatically: YYYYMMDD_NNN_schematic.json.

This is the primary save format. Store it in your project folder and commit it to GitHub.

Auto-save There is no auto-save. Export JSON before closing the browser or navigating away.

18 Import JSON

Clicking IMPORT opens a file picker. Select a previously exported .json schematic file. The canvas loads with all components and wires restored. The view fits to the loaded content automatically.

Import replaces the current canvas completely. Export first if you want to keep the current state.

19 Export SVG

Clicking EXPORT SVG downloads a vector schematic suitable for documentation, printing, or embedding in reports. The SVG includes all wires and component labels.

Note: SVG export captures the schematic geometry and text. It does not include interactive elements. For a full state snapshot, use Export JSON.

23 Troubleshooting

Components not snapping to grid

Zoom in. At very low zoom levels the grid spacing is smaller than the cursor movement precision. Work at 50% zoom or higher for accurate placement.

Wire not connecting to pin

Pin snapping requires the cursor to be within 15 screen pixels of the pin. Zoom in to the pin before starting or ending the wire.

Schematic lost on page reload

Export JSON before closing or reloading. There is no browser-side persistence in the current version. GitHub Sync is on the roadmap.

Export SVG looks blank

SVG export requires at least one component on the canvas. Place a component before exporting.

Library panel not visible on mobile

Scroll the page left, or use the search field at the top of the library. On very small screens the library collapses — tap the left edge to reveal it.
